# 🏦 Banking Management System (Python OOP Project)

## 📌 Overview

This project is a **console-based Banking Management System** built using Python and Object-Oriented Programming principles.
It simulates how a real banking backend works, including user authentication, account management, and transaction handling.

The system is designed using a **modular layered architecture**, separating authentication, business logic, data models, and utilities — similar to real-world software systems.

---

## 🚀 Features

* User Signup & Login system
* Unique username validation
* Automatic account number generation
* Support for multiple users
* Savings and Current account types
* Transaction history logs
* Deposit & Withdraw functionality
* Balance checking
* JSON-based persistent storage
* Modular folder structure
* Clean OOP architecture

## 🗂 Project Structure

```
bank_project/
│
├── models/        # Account classes & data models
├── services/      # Business logic layer
├── utils/         # Helper functions
├── data/          # JSON storage files
└── main.py        # Entry point
```

---

## ⚙️ How It Works

1. User signs up or logs in
2. System authenticates user
3. User creates bank account
4. System generates unique account number
5. User performs transactions
6. Data is saved to JSON file

---

## ▶️ How to Run

```bash
python main.py
```
